# 🧭 Browser History App

A simple and interactive **Browser History Application** built using **React.js**. This app allows users to view, search, and delete browsing history items in real-time.

---

## 🚀 Features

* 🔍 **Search Functionality**

  * Filter history items based on user input
  * Case-insensitive search

* 🗑️ **Delete Functionality**

  * Remove individual history items instantly

* 📭 **Empty State**

  * Displays *"There is no history to show"* when no results are found

* 📋 **Dynamic Rendering**

  * Displays list of history items using React state

## 📸 Functionality Overview

* Displays a list of browser history items
* Each item contains:

  * Time accessed
  * Website logo
  * Title
  * Domain URL
* Users can:

  * Search history using the search bar
  * Delete any history item

## ▶️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/browser-history-app.git
   ```

2. Navigate to the project folder:

   ```bash
   cd browser-history-app
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the app:

   ```bash
   npm start
   ```
